## Non-obvious constraints for implementers

1. **`_djb2_hash` / `_int_to_base36`:** Must match `djb2Hash` + `Math.abs(...).toString(36)` from TS — not hex, not classic djb2(5381).
2. **`find_session` vs `find_project_dir`:** Do not merge these code paths conceptually — session ID search is global; target directory for a **given** absolute path uses `find_project_dir`.
3. **Golden tests:** Node one-liner in plan uses compiled `sessionStoragePortable.js` — ensure build path exists or invoke TS via project’s normal compile/test harness.
4. **NFC:** Worktree paths in TS use NFC normalization; v0.8 documents optional `unicodedata.normalize('NFC', ...)` for parity on non-ASCII paths — Linux ASCII common case often unchanged.
